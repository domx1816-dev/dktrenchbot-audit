## Quick Deploy (Recommended)

### Cloudflare Pages — 2 Minutes
1. Visit https://pages.cloudflare.com/
2. Click **"Create project"** → **"Upload assets"**
3. Drag this entire `dktrenchbot-audit` folder into the upload area
4. Name it: `dktrenchbot-audit`
5. Click **Deploy**
6. Share the URL (e.g., `https://dktrenchbot-audit.pages.dev`)

### GitHub Pages — 5 Minutes
1. Create new repo: `github.com/new`
2. Upload `index.html` to the repo
3. Settings → Pages → Deploy from `main` branch
4. URL: `https://yourusername.github.io/repo-name/`
